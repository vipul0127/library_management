Librarian Menu
As a librarian, you can perform the following actions:

Register a member: Register a new library member.
Remove a member: Remove a member from the library.
Add a book: Add a new book to the library's collection.
Remove a book: Remove a book from the library's collection.
View all members along with their books and fines: Display a list of library members, their borrowed books, and fines.
View all books: Display a list of all available books.
Member Menu
As a member, you can perform the following actions:

List Available Books: Display a list of books available for borrowing.
List My Books: Display a list of books you have borrowed.
Issue a Book: Borrow a book from the library.
Return a Book: Return a borrowed book to the library.
Pay Fine: Pay any fines accrued for late returns.
Important Notes
Librarians and members should log in with their name and phone number.
Members can only borrow up to two books at a time.
Fines are imposed for late returns (more than 10 seconds).
Fines can be paid by members.
